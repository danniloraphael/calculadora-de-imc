# Calculadora de IMC

## Descrição
Calculadora de Índice de Massa Corporal (IMC) simples desenvolvida em JavaScript, HTML e CSS.

## Como Usar
1. Insira seu peso (em kg)
2. Insira sua altura (em m)
3. Clique em calcular
4. Veja seu IMC e classificação

## Fórmula
```
IMC = peso / (altura²)
```

## Classificação
- Abaixo de 18,5: Abaixo do peso
- 18,5 - 24,9: Peso normal
- 25 - 29,9: Sobrepeso
- 30 - 34,9: Obesidade  grau I
- 35 - 39,9: Obesidade grau II
- Acima de 40: Obesidade grave
